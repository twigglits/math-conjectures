The [ErdЕ‘s-Straus conjecture](https://en.wikipedia.org/wiki/Erd%C5%91s%E2%80%93Straus_conjecture). Perhaps the first place it appears in the literature is in a paper of OblГЎth [\[Ob50\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) (submitted in 1948), which describes it as a conjecture of ErdЕ‘s.

The existence of a representation of 4/рқ‘ӣ as the sum of at most four distinct unit fractions follows trivially from a greedy algorithm.

Schinzel conjectured (see [\[Si56\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242)) the generalisation that, for any fixed рқ‘Һ, if рқ‘ӣ is sufficiently large in terms of рқ‘Һ then there exist distinct integers 1вүӨрқ‘Ҙ<рқ‘Ұ<рқ‘§ such thatрқ‘Һрқ‘ӣ=1рқ‘Ҙ+1рқ‘Ұ+1рқ‘§.When рқ‘Һ=5 this conjecture is due to SierpiЕ„ski [\[Si56\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242). For more background and results on this generalisation see Pomerance and Weingartner [\[PoWe25\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242).

It suffices to prove this when рқ‘ӣ is prime. This has been verified for all рқ‘ӣвүӨ1018 [\[MiDu25\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242).

There are many partial results, some of which are listed below.

- OblГЎth [\[Ob50\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) noted it is true if рқ‘ӣ+1 is divisible by a prime вүЎ3вҒў(modвҒЎ4). This implies almost all рқ‘ӣ have the required decomposition.
- Arguing via parametric solutions, Mordell [\[Mo69\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) proved it is true for all рқ‘ӣ except those congruent to one of {1,121,169,289,361,529} modulo 840.
- Terzi [\[Te71\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) extended this to prove that it is true for all рқ‘ӣ except those congruent to one of 198 possible bad congruences modulo 120120.
- Vaughan [\[Va70\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) proved that the number of exceptions in \[1,рқ‘Ҙ\] isвүӨрқ‘ҘвҒўexpвҒЎ(вҲ’рқ‘җвҒў(logвҒЎрқ‘Ҙ)2/3)for some constant рқ‘җ>0.
- This conjecture is equivalent (see Theorem 1 of [\[BlEl22\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242)) to the statement that, for any prime рқ‘қ, there exist integers рқ‘Һ,рқ‘җ,рқ‘‘вүҘ1 such that either рқ‘қвүЎвҲ’рқ‘Һ/рқ‘җвҒў(modвҒЎ4вҒўрқ‘ҺвҒўрқ‘җвҒўрқ‘‘вҲ’1) or рқ‘қвүЎвҲ’4вҒўрқ‘җ2вҒўрқ‘‘+1рқ‘ҳвҒў(modвҒЎ4вҒўрқ‘җвҒўрқ‘‘) for some рқ‘ҳвҲЈ4вҒўрқ‘җ2вҒўрқ‘‘+1.
- Bright and Loughran [\[BrLo20\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) have shown there is no Brauer-Manin obstruction to the existence of solutions.
- If рқ‘“вҒЎ(рқ‘ӣ) counts the number of solutions then Elsholtz and Tao [\[ElTa13\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) have provedвҲ‘рқ‘қвүӨрқ‘Ғрқ‘“вҒЎ(рқ‘қ)=рқ‘ҒвҒў(logвҒЎрқ‘Ғ)2+рқ‘ңвҒЎ(1)and рқ‘“вҒЎ(рқ‘қ)вүӨрқ‘қ3/5+рқ‘ңвҒЎ(1) for all primes рқ‘қ.
- Elsholtz and Planitzer [\[ElPl20\]](https://www.erdosproblems.com/forum/thread/242?order=oldest#bib-container242) have proved that for almost all рқ‘ӣрқ‘“вҒЎ(рқ‘ӣ)вүҘ(logвҒЎрқ‘ӣ)logвҒЎ6+рқ‘ңвҒЎ(1).
